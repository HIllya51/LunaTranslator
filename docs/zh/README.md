## 启动

下载后解压到任意目录

**LunaTranslator.exe** 会以普通模式启动 

**LunaTranslator_admin.exe** 会以管理员权限启动，部分游戏需要管理员权限才能HOOK，仅这时需要使用这个，其他时候普通模式启动即可。

**LunaTranslator_debug.exe** 会显示命令行窗口


::: danger
有时会被杀毒软件杀掉，请添加信任并重新下载解压
:::
