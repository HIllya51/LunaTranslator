# English Word Lemmatization

> [!TIP]
> The primary purpose of this extension is to enable looking up words using their base forms during dictionary searches.

1. ## Download Dependencies and Extract
    
    Download the [Analyzer](https://lunatranslator.org/Resource/otherresource/spacy_wrapper) and extract the contents into the software's **root directory**, **LunaTranslator directory**, or **configuration directory (userconfig)**.

1. ## Switch Language

    Switch the source language to **English**.
